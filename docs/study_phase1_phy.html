OFDM Symbol Structure & Subcarrier Spacing

Prereq: basic RF — path loss, dBm, RSSI. No earlier days required.
Socratic Question
If 802.11ax uses 78.125 kHz subcarrier spacing instead of 312.5 kHz (11ac), why does the symbol time grow 4×, and what does that buy you in a dense multipath environment?
OFDM uses a cyclic prefix to absorb multipath delay spread. If you lengthen the symbol, what happens to the CP overhead as a percentage — and when does that trade-off break down?
Reading (T1 Corpus)

80211-2024.pdf §17 — OFDM PHY (subcarrier count, spacing, symbol duration table across amendments)
802.11be - Wi-Fi 7.pdf — EHT PHY section, subcarrier comparison table

Lab Exercise
AP1 (AP3000) — Beacon Capture + HE Capabilities IE Decode
SSH to AP1 → run capture interface wifi0 duration 30 → pull PCAP to MacBook → open in Wireshark → filter wlan.fc.type_subtype == 0x08 (beacons) → expand HE Capabilities IE and EHT Capabilities IE → record: subcarrier spacing, max NSS, supported channel widths.

Compare to show wifi0 radio output on AP1 — verify IE matches runtime config.
Close-Out Check

Can you explain in one sentence why 802.11ax chose narrower subcarriers without reading notes?

MCS Index Mechanics — SNR Floor, Cell-Edge Collapse

Prereq: Day 1 (OFDM symbol structure)
Socratic Questions
MCS 11 (1024-QAM) needs ~35 dB SNR; MCS 7 (64-QAM) needs ~25 dB. Why does doubling the constellation density cost 10 dB SNR — what's the fundamental coding-gain limit?
In a cell-edge design, if your minimum RSSI is −72 dBm and noise floor is −95 dBm, what's the usable MCS ceiling — and what does the MCS step-down pattern look like as a client walks away from the AP?
Reading (T1 Corpus)

802.11ac - VHT - MCS, SNR and RSSI.pdf — full MCS table with minimum SNR per stream
dB Math in 5 Minutes Handout.pdf — SNR arithmetic, noise floor formula

Lab Exercise
iPerf3 MCS Walk Test
Run iPerf3 server on a wired host connected to SW1 → run iPerf3 client on MacBook at three positions (1m, 5m, far corner) → after each run, capture show wifi station <mac> on AP1 to record negotiated MCS and SS count → plot RSSI vs MCS vs throughput in a simple table.

MU-MIMO vs OFDMA — When Each Wins

Prereq: Day 1–2 (OFDM, MCS)
Socratic Questions
A meeting room has 30 phones each doing 100 kbps voice (small, frequent frames). MU-MIMO or OFDMA — which scheduler wins, and why does the frame size matter more than the client count here?
DL MU-MIMO sends different data to multiple clients simultaneously using spatial separation. DL OFDMA sends different data on different subcarrier groups. For a client doing a 4 MB file download, which gives more peak throughput and why?
