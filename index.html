802.11 Mastery

A six-phase Socratic curriculum — from OFDM internals to field-ready capacity design — with live lab exercises on AP3000 + 5320 EXOS.

Phases Schedule Hub Phase 1 — PHY Phase 2 — MAC Phase 3 — Security Phase 4 — QoS Phase 5 — Capacity Phase 6 — Capstone References

This curriculum assumes you already own WPA2/WPA3 4-way handshake forensics, PEAP-MSCHAPv2 architecture, and DHCP cascade debugging. Each phase opens one layer deeper — starting at OFDM symbol structure, not "what is Wi-Fi."

Every day = 1–2 hour session. Every session = Socratic questions → WLPC corpus reading → AP3000 lab capture → HTML knowledge note. Pick up any day; each section is self-contained with prereq tags.
